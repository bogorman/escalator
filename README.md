# eSCALAtor

Opinionated Full stack Scala starter project. Scala with some guide rails.

- Scala 2.13
- Akka + AkkaHttp
- Laminar
- Postgres
- Quill
- Monix
- Generate scaffolding from DB. 
- Flyway migrations. 
- Simple starter app with Login/User Registration. 
- UI / Backend share models. 
- Monitoring with Kamon
